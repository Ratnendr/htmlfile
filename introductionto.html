THE B TAG

The i tag

The i tag is often used to indicate a technical term ,a phrase only when there is not a more appropriate semantic element,such as em (emphasized text).


The u tag

The underline tag or u tag in html help to misspelled works. We can write it inside the start tag u and closing tag u to underline the gtext.


The select tag

The select tag is used to create drop-down list.The element most often is used in form , to collect user input.The name attribute is needed to reference the form data after the form is submitted.


The button tag

the button tag defines a clickable.Inside a button element you can put text (and tags like (i),(b),(strong),(br),(img),etc).That is not possible with a button with the input element.


The br tag

The br tag is inserts a single line breaks. The br tag is usful for writting addresses or poem.The br tag is an empty tag which means that it has no end tag.
